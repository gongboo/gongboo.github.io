C언어는 크기와 용도가 다른 다양한 자료형이 있다. 

변수: 데이터 값을 저장하는 저장공간. 변수 이름, 변수의 자료형, 변수의 저장값으로 구성
```c
int word; //변수 선언
word=123; //변수 초기화(선언과 동시에 할 수 있고 안하면 쓰레기값 저장)
```

상수:변수처럼 바뀔 수 없는 고정된 값.
1,'a' 이런거도 되지만

```c
const int word=123; //이렇게 하면 고정된 변수 즉 상수로 만들 수 있다.
```
